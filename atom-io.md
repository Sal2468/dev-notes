Atom Editor Notes
=================

## Command Cheatsheet

* `Ctl + Shift + P` -- open the command list (the first shortcut to learn)
* `Ctl + Shift + M` -- open/close an HTML preview of a Markdown doc (emojis included)
* `cmd + alt + Z` -- checks out the HEAD revision of the file in the editor

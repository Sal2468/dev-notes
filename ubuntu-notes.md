Ubuntu Tips
-----------

+ create a new Desktop icon

`sudo sublime /usr/share/applications/sublime.desktop`

+ install Java7

<verbatim>
sudo add-apt-repository ppa:webupd8team/java  
sudo apt-get update  
sudo apt-get install oracle-java7-installer  
</verbatim>
